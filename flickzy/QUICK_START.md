# Flickzy Quick Start Guide

## üöÄ Get Started in 30 Seconds

### Create a Room (No Login Required!)
1. Go to **Movies**, **Music**, or **Games** page
2. Click the **"Create Room"** card
3. Enter a room name
4. Click **"Create Room"** button
5. **Share the 6-character room code** with friends!

### Join a Room (No Login Required!)
1. Get a room code from a friend
2. Go to **Movies**, **Music**, or **Games** page
3. Click the **"Join Room"** card
4. Enter the room code
5. Click **"Join Room"** button
6. **You're in!**

## ‚ùì FAQ

### Do I need to create an account?
**No!** You can use all features without logging in. Login is optional and only provides username display and participant tracking.

### How do I share a room?
Just share the 6-character room code with anyone. They can enter it on the "Join Room" dialog.

### Can I use this on mobile?
**Yes!** Flickzy works on all devices - desktop, tablet, and mobile.

### How many people can join a room?
There's no hard limit, but video calls work best with 2-4 participants.

### Do rooms expire?
Rooms stay active as long as someone is in them. Inactive rooms may be cleaned up periodically.

### Can I password-protect a room?
Not yet, but it's on the roadmap! For now, only share codes with people you trust.

### What if someone is being disruptive?
Currently, there's no kick/ban feature. Consider creating a new room with a different code.

### Can I save my favorite rooms?
Not yet for anonymous users. Log in to get access to room history and favorites (coming soon).

### What browsers are supported?
All modern browsers: Chrome, Firefox, Safari, Edge. WebRTC features require camera/mic permissions.

### Is my data private?
Room codes are randomly generated. Messages are visible to all room participants. User profiles (if logged in) are private.

## üÜò Troubleshooting

### Room not found
- Check the room code is correct (6 characters)
- Room might have been deleted
- Try creating a new room

### Video not loading
- Check the URL is valid
- Try a different video
- Check your internet connection

### Chat not working
- Refresh the page
- Check browser console for errors
- Try in incognito mode

### Video call not working
- Allow camera/mic permissions
- Check your camera/mic are working
- Try a different browser
- Check firewall settings
